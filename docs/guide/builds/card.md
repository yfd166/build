## 功能简介

![image-20230707094950995](https://test-md.obs.cn-south-1.myhuaweicloud.com/img/apple/202307070949162.png)

证件申请模块提供施工证件及布撤展车辆证件的信息申报功能。若您申报的项目开启了此模块，则应在购买证件后，转到此模块申报证件信息。

::: waring 注意事项

证件请谨慎购买，一经付款概不退款！

:::
