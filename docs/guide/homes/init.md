## 快速初始化系统

1. 添加展馆

展馆名称、地点一页搞定。

2. 添加工作人员

收集除姓名、职位等必填信息外，还可添加手机号、办公电话、职责范围等辅助说明信息。

3. 设置模板资料

输入模板资料名称、类型、支持的文件格式、说明资料上传要求、模板文件。

4. 创建租赁物品

   - 添加租赁品类

   - 创建物品，设置规格及单价。

完成以上步骤后，即可开始创建项目。
